# BellaChess Web Application

This is the Next.js web application for BellaChess - a comprehensive chess platform that combines AI-powered learning, etude-based training, and Web3 collectibles.

## Overview

BellaChess is a modern chess platform that includes:

- **AI Coach**: Personalized feedback and game analysis
- **Etude Challenges**: Progressive tactical puzzles
- **NFT Collectibles**: Digital trophies earned through achievements
- **Cross-platform**: Web, iOS, and Android support

## Tech Stack

- **Framework**: Next.js 14+ with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **AI**: Stockfish WebAssembly integration
- **Web3**: Polygon integration for NFTs

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
src/
├── app/              # Next.js App Router pages
├── components/       # Reusable UI components
├── lib/              # Utilities and helper functions
└── styles/           # Global styles
```
